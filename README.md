# Сайт юридической компании «АСТРЕНТ» (Ярославль)

Продающий адаптивный сайт на чистом HTML/CSS/JS — без сборки и зависимостей.
Открывается двойным кликом по `index.html` и работает на любом хостинге.
Каналы связи настроены под российский рынок: **MAX + ВКонтакте** (без WhatsApp/Telegram).

## Структура
```
reshenie-100/
├── index.html      # разметка и весь контент
├── css/styles.css  # стили (палитра «петроль + айвори + латунь»)
├── js/main.js      # меню, счётчики, аккордеон, анимации, формы, маска телефона
├── images/         # сюда положите реальные фото (см. ниже)
└── README.md
```

## Запуск локально
```bash
cd reshenie-100
python3 -m http.server 8123
# открыть http://localhost:8123
```

## 🎨 Цветовая гамма (плюсы/минусы — как просили)
Вы просили увести от «слишком красного, который бьёт по глазам».

| Палитра | Плюсы | Минусы | Решение |
|---------|-------|--------|---------|
| Синий + яркое золото (1-я версия) | авторитет, классика | холодновато, банально | заменено |
| **Петроль (сине-зелёный) + айвори + приглушённая латунь** | спокойная, премиальная, мягкая для глаз, выделяет среди конкурентов | зелёный реже у «классики» | **выбрано ✅** |
| Бордо/винный + крем | дорого, солидно | красно-винный диапазон, который вы просили избегать | отклонено |

Цвета — в `:root` файла `css/styles.css`:
`--navy #123a4b` · `--gold #c2954f` · `--bg-alt #f5f1e8` · `--max #2683ff` · `--vk #0077ff`.
